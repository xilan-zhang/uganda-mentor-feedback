# Business Profiles

Personalized business profile pages for a Stanford research study.

**Browse all profiles:** [https://xilan-zhang.github.io/uganda-mentor-feedback/](https://xilan-zhang.github.io/uganda-mentor-feedback/)

Each participant has a unique page (P001–P100) with 8 business profiles to review.

## Direct links

To go directly to a participant's page, use:

```
https://xilan-zhang.github.io/uganda-mentor-feedback/P001.html
```

Replace `P001` with the participant ID.
